  - [x] id 값이 없는 경우 예외를 발생시킨다.
  - [x] 예약을 삭제할 때 데이터베이스에 값을 삭제한다.

```

3 방탈출 예약 추가
4 방탈출 예약시간 추가
5 방탈출 예약 삭제
6 방탈출 예약시간 삭제

방탈출 예약 내역입니다.
예약번호      예약자         날짜          시간

순서         시간           
1           10:00:00
```
